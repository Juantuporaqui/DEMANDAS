# Documentos PDF - Caso Quart (ETJ 1428/2025)

## Cómo subir PDFs

1. Sube los archivos PDF a esta carpeta
2. Nombra los archivos con el formato: `Doc_XX_Descripcion.pdf`
3. Haz commit y push
4. Los PDFs aparecerán automáticamente en la pestaña Documentos

## Ejemplo de nombres:
- `Sentencia_362_2023_Divorcio.pdf`
- `Demanda_Ejecucion.pdf`
- `Oposicion_Juan.pdf`
- `Impugnacion_Vicenta.pdf`
