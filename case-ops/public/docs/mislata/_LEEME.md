# Documentos PDF - Caso Mislata (J.V. 1185/2025)

## Cómo subir PDFs

1. Sube los archivos PDF a esta carpeta
2. Nombra los archivos con el formato: `Doc_XX_Descripcion.pdf`
3. Haz commit y push
4. Los PDFs aparecerán automáticamente en la pestaña Documentos

## Ejemplo de nombres:
- `Demanda_Reclamacion.pdf`
- `Contestacion_Vicenta.pdf`
- `Extractos_Hipoteca.pdf`
