# Documentos PDF - Caso Picassent (P.O. 715/2024)

## Cómo subir PDFs

1. Sube los archivos PDF a esta carpeta
2. Nombra los archivos con el formato: `Doc_XX_Descripcion.pdf`
3. Haz commit y push
4. Los PDFs aparecerán automáticamente en la pestaña Documentos

## Ejemplo de nombres:
- `Doc_01_Demanda.pdf`
- `Doc_02_Contestacion.pdf`
- `Doc_13_Justificante_Ingreso_18000.pdf`
- `Doc_53_Escritura_Hipoteca.pdf`
